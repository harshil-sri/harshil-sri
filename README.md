### ⚡ About Me

I'm an **AIML Student at VIPS-TC** who treats every project like a domain expansion. I love fusing ML with real engineering — whether that's building competitive arenas powered by NLP models, or reverse-engineering my own biology with data pipelines.

- 🥊 Currently deep in: **Code Kaisen** — a 1v1 AI-judged coding battleground inspired by Jujutsu Kaisen
- 🧬 Also building: **Project Apotheosis** — a personalized bio-AI engine to mathematically optimize physical performance  
- 🌱 Currently learning: **Computer Vision**, **Deep Learning**, and leveling up in **Full-Stack** (React, Node, Express)
- ⚡ Fun fact: I once wrote a Python script to force my lightbulbs to appreciate Joji albums. It worked.
